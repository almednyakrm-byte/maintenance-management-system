# نظام إدارة مصمم لتقديم خدمات الصيانة (الفواتير، جداول التقدم، مراقبة الوقت)
===========================================================

## ملخص المشروع
-----------------

نظام إدارة مصمم لتقديم خدمات الصيانة، يتيح للمستخدمين إدارة الفواتير والتقدم والوقت بشكل فعال.

## وظيفة المشروع
-----------------

* إدارة الفواتير: إضافة وتعديل وإزالة الفواتير
* جداول التقدم: إضافة وتعديل وإزالة جداول التقدم
* مراقبة الوقت: إضافة وتعديل وإزالة مراقبة الوقت

## هيكل المشروع
-----------------

* `docker-compose.yml`: هيكل المشروع باستخدام docker-compose
* `app/`: مجلد المشروع الرئيسي
 + `models/`: مجلد الموديلات (tables)
 + `controllers/`: مجلد التحكم (controllers)
 + `routes/`: مجلد الطرق (routes)
* `docker/`: مجلد docker
 + `Dockerfile`: هيكل docker
 + `docker-compose.yml`: هيكل docker-compose

## إعداد البيئة باستخدام docker-compose
-----------------------------------------

1. تأكد من أن docker-compose مثبت على جهازك.
2. افتح مجلد المشروع في terminal.
3. استخدم الأمر التالي لتشغيل البيئة:
bash
docker-compose up

4. افتح متصفحك ومرر إلى `http://localhost:8000` لفتح تطبيق النظام.

## وحدات المشروع
-----------------

### وحدات الموديلات

* `فواتير`: تجمع الفواتير
* `جدول التقدم`: تجمع جداول التقدم
* `مراقبة الوقت`: تجمع مراقبة الوقت

### وحدات التحكم

* `Controller_Fatoura`: التحكم في فواتير
* `Controller_Jadwal`: التحكم في جداول التقدم
* `Controller_Maraba`: التحكم في مراقبة الوقت

### أدوار النظام

* `admin`: الدور الرئيسي
* `user`: الدور المستخدم
